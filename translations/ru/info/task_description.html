This mission is like a subset of Эта миссия является разновидностью другой миссии под названием “Visibilities”.

Вам предоставляется список строк в виде сетки. Пустые ячейки будут представлены '.', а блоки - 'X'. Блоки не примыкают друг к другу ни вертикально, ни горизонтально.

Одна область состоит из пустых ячеек, смежных друг с другом по вертикали и по горизонтали. Вы должны вернуть координаты блоков, которые составляют границу.

Примеры: 

set(boundary_blocks(['..X',
                     '.X.',
                     'X..'])) == {(0, 2), (1, 1), (2, 0)}
set(boundary_blocks(['...',
                     '.X.',
                     'X..'])) == set()
set(boundary_blocks(['X.X.',
                     '.X..',
                     '..X.',
                     '....'])) == {(0, 0), (0, 2), (1, 1)}

Входные данные: Список строк.

Выходные данные: Итерируемые кортежи из двух целых чисел (В порядке строк, столбцов).

Предварительные условия:

  • 3 ≤ len(grid) ≤ 20
  • 3 ≤ len(grid[0]) ≤ 20
  • all(set(row) <= {'.', 'X'} for row in grid)
